6

Common Mistakes

These are the alt text errors we see most often in audits.

01

Missing alt entirely

No alt attribute at all — screen readers read the file name instead

02

Keyword stuffing

Using alt text for SEO keywords instead of genuine descriptions

03

Starting with "image of" or "photo of"

Screen readers already announce it's an image — don't repeat it

04

Same alt for every image

Generic "product image" repeated 50 times helps no one
