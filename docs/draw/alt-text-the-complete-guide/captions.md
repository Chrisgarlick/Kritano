## Slide 2 — What Is Alt Text?

**Caption:**
Alt text is a short description added to an image's HTML. Screen readers read it aloud. Search engines use it to index. And it displays when images can't load.

Three audiences. One attribute.

## Slide 4 — Good vs Bad Examples

**Caption:**
alt="image" ← bad.
alt="IMG_4392.jpg" ← worse.
alt="Golden retriever sitting in a sunny park" ← good.

Be specific. Be brief. Never start with "image of" or "photo of" — screen readers already announce it's an image.

## Slide 5 — Decorative vs Meaningful

**Caption:**
Not every image needs a description.

Product photos, charts, team photos → describe them.
Background textures, decorative dividers, spacer images → use alt="" (empty).

The key question: does this image convey information?

## Slide 6 — Decision Flowchart

**Caption:**
The alt text decision flowchart:

1. Does it convey information? No → alt=""
2. Is it a link/button? → describe the action
3. Is it complex (chart/diagram)? → short alt + longer description
4. Otherwise → describe the content in under 125 characters

## Slide 7 — Common Mistakes

**Caption:**
The 4 alt text mistakes we see most in audits:

01. Missing alt entirely — screen readers read the file name
02. Keyword stuffing — alt is for users, not SEO tricks
03. Starting with "image of" — redundant
04. Same alt for every image — lazy and unhelpful
