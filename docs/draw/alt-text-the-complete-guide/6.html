The Decision Flowchart

Ask yourself these questions for every image.

1

Does the image convey information?

If no → use alt=""

2

Is it a functional image (link, button)?

If yes → describe the action, not the image

3

Is it a complex image (chart, diagram)?

If yes → short alt + longer description nearby

4

Otherwise → describe the content

Keep it under 125 characters. Be specific.
