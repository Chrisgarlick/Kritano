What Is Alt Text?

A short text description added to an image's HTML. It's read aloud by screen readers and displayed when images can't load.

<img
  src="team-photo.jpg"
  alt="PagePulser team working at
        their desks in the office"
/>

For users

Screen readers read it aloud to visually impaired users

For SEO

Search engines use it to understand image content

For fallback

Displays when images fail to load on slow connections
