SSL / HTTPS

The padlock in the address bar. Without it, everything your users send is visible to anyone on the network.

What Can Go Wrong

Expired SSL certificate — Chrome shows a full-screen warning
No automatic HTTP → HTTPS redirect
Login forms served over plain HTTP
Fix

Install a valid SSL certificate (free via Let's Encrypt). Force HTTPS with a 301 redirect. Set HSTS headers to prevent downgrade attacks.

82% of users will leave a site that shows a "Not Secure" warning
