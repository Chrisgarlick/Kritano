Poor Colour Contrast

Low contrast text is unreadable for users with visual impairments — and annoying for everyone else. WCAG requires a minimum ratio of 4.5:1 for body text.

Fail Light text 1.9:1 ratio
Pass Dark text 12.6:1 ratio
Quick Fix

Test every text/background combination against WCAG AA. Use a contrast checker tool — aim for 4.5:1 minimum on body text, 3:1 on large text.
