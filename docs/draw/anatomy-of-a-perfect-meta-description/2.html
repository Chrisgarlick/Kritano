The Formula

Keyword + Value + Specific Details + Call to Action

Do

+ Front-load your target keyword
+ Include a specific benefit or number
+ End with a clear CTA
+ Write unique descriptions per page

Don't

- Stuff keywords unnaturally
- Go over 155 characters
- Copy the same description to all pages
- Leave it blank (Google will guess)
