# Captions — Anatomy of a Perfect Meta Description

## Variation 1 — Light with SERP Mockup

**Caption:**
155 characters. That's all you get to convince someone to click your link instead of the other nine on the page.

Most sites either leave their meta description blank (letting Google guess) or stuff it with keywords. Neither works.

Here's the anatomy of one that does — with a real Google SERP mockup showing exactly what users see:

1. Lead with the value + keyword
2. Be specific about what's included
3. End with a clear call to action
4. Stay under 155 characters

## Variation 3 — Brand Indigo Cheat Sheet

**Caption:**
Meta description cheat sheet. Save this.

→ 155 chars max (desktop), 120 on mobile
→ Front-load your keyword in the first 10 words
→ Include one specific number or timeframe
→ End with an action verb
→ Write a unique description for every page

Pages with no meta description let Google guess. And Google guesses badly.
