Logo Usage Guidelines

Primary Logo

Use "The Watcher" (#1) or "The Double Seer" (#10) as the primary logo for website headers, marketing materials, and brand communications.

Icon Mark

For favicons and app icons, use just the animated "i" dot element (the eye) from any design as a standalone icon.

Animation Context

Use animated versions for digital contexts. Static versions (freeze first frame) for print and static images.

Color Variations

On light backgrounds, use full color. On dark backgrounds, use as shown. Monochrome white/black for special cases.
