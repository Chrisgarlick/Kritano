# Kritano Brand Identity

## 1. Brand Name Usage

### Correct Usage
- **Written form**: Kritano (capital K, lowercase remainder, no spaces or hyphens)
- **Code references**: `kritano` (all lowercase), `Kritano` (PascalCase for class names/components)
- **Domain**: kritano.com

### Incorrect Usage
- ~~kritano~~ (missing capital in prose)
- ~~KRITANO~~ (all caps -- not an acronym)
- ~~Krit-ano~~ (hyphenated)
- ~~KritAno~~ (camelCase -- it is not two words)
- ~~Krit Ano~~ (two words)

### Pronunciation
**kri-TAH-no** -- stress on the second syllable. Rhymes with "Milano."

### Origin Story (for About page)
Kritano takes its name from the Greek *krites*, meaning judge or evaluator -- reflecting the platform's purpose of delivering clear, authoritative judgment on every dimension of your website's health.

## 3. Brand Voice

### Tone Keywords
1. **Authoritative** -- speaks with earned confidence
2. **Precise** -- no filler, no hedging, every word carries weight
3. **Candid** -- tells you what is wrong without softening it into uselessness
4. **Practical** -- every finding includes a path forward
5. **Measured** -- calm expertise, not hype

### Brand Archetype
**Primary:** Sage (wisdom, expertise, clarity)
**Secondary:** Judge (authority, evaluation, verdict)

### Example Voice

**Hero copy:**
"Kritano audits your website across six categories and returns a clear verdict -- what is working, what is failing, and what to fix first."

**Post-audit:**
"Kritano found 47 unique issues across your site. Here is where to start."

**Monitoring:**
"Your website's full judgment -- delivered in minutes, monitored on schedule."

## 5. Crawler/Bot Identity

### User Agent String
```
KritanoBot/1.0 (+https://kritano.com/bot)
```

### Bot Description
KritanoBot is the web crawler operated by Kritano (https://kritano.com). It accesses pages solely to perform website health audits requested by verified site owners. KritanoBot respects robots.txt directives, does not index or store page content beyond audit analysis, and identifies itself with the user agent string `KritanoBot/1.0`. For questions or to report issues, contact support@kritano.com.

## 8. Visual Identity

**No changes from existing Kritano brand guidelines.** The following are preserved exactly:

- **Primary colour:** Indigo-600 (`#4F46E5`)
- **Accent colour:** Amber palette
- **Display font:** Instrument Serif
- **Body font:** Outfit
- **Code font:** JetBrains Mono
- **Spacing:** 4px base unit, 8px primary scale
- **Components:** All button, card, input, and focus state patterns unchanged
- **Semantic colours:** All status and severity colour mappings unchanged

The only visual change is the logo text: "Kritano" becomes "Kritano" using the same Instrument Serif display font.
